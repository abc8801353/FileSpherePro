# FileSphere Pro v1.0.0 🎉

**文件星球专业版** - 正式版本发布！

## ✨ 核心功能

### 📝 批量重命名
- 支持统一前缀、自动编号、字符替换
- 实时预览重命名效果
- 批量去除空格和特殊字符

### 📁 文件分类整理
- 按文件类型自动分类（文档、图片、视频等）
- 支持自定义关键词规则分类
- 一键归档到指定文件夹

### 🔍 智能去重
- 精确识别重复文件（基于 MD5）
- 多种保留策略（最新、最旧、最大、最小）
- 支持移至回收站或永久删除

### 📄 内容提取
- 批量提取 Word、PDF、Excel、TXT 文件内容
- 支持多种编码格式
- 导出为单个文本文件

### 📊 Excel 合并
- 一键合并多个 Excel 文件
- 保留原始格式和数据
- 自动生成合并报告

### 📈 文件统计分析
- 可视化文件类型分布
- 文件大小统计
- 导出统计报告

## 🌐 多语言支持
- 简体中文
- 繁體中文
- English

## 🎨 界面特性
- 现代化 Fluent 设计
- 支持浅色/深色主题
- 响应式布局

## 🔒 系统特性
- 单实例运行（防止多开）
- 系统托盘支持
- 快捷键操作

## 📋 系统要求
- Windows 10/11 (64-bit)
- 无需安装 .NET 运行时（独立发布）

## 📦 安装说明
下载 `FileSpherePro_Setup_v1.0.0.exe` 后直接运行即可。
